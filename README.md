# Dictionary Web Application
## Description
This is a simple web application that allows users to search for english words from an external API.
## Features
- A form to search the word
- Fetch data from the API
- Search for words
- Display the word's pronunciation, an audio of pronunciation, definitions and synonyms of the words searched.
- Error handling for invalid inputs
## How to Run the Project
- Fork and Clone the repository
- Open the folder
- Run the HTML file on the browser
## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
